📝 说明

  • 价格单位为 $/M（每百万 tokens 的美元价）
  • 关闭「启用自定义价格」或关闭某一行的「启用」后，该范围使用会话中的 OpenClaw 账面成本
  • 开启全局且开启对应行时，使用自定义单价重算成本（理论价）
  • 参考表基于 models.json（$OPENCLAW_CONFIG_DIR 或默认 ~/.openclaw 下 agents/main/agent/models.json）中带有效单价的模型
  • Cache Read/Write 留空表示不设单独缓存价：按 Input / Output 原价计算（读取用 Input 单价，写入用 Output 单价）
  • 保存价格配置或切换开关后会自动失效缓存，重新计算所有成本
